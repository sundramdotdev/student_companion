# Roadmap

This roadmap outlines the past, present, and future trajectory of Student Companion. 

## Phase 1: Foundation (Completed)
- [x] Initial Project Setup & Clean Architecture Implementation
- [x] Hive Database Configuration for Offline Storage
- [x] Core Feature: Subject & Attendance Tracking
- [x] Core Feature: Timetable Scheduling
- [x] Core Feature: GPA Calculation (Semester & Cumulative)
- [x] Custom Grading Scales support

## Phase 2: Polish & Prediction (Completed)
- [x] Material 3 UI/UX Redesign
- [x] "Bunk Calculator" (Predictive Attendance Analytics)
- [x] Internal Marks Tracking
- [x] Dynamic Theme Toggle (Light/Dark/System)
- [x] Centralized Version Management (Package Info Plus)
- [x] Open-Source Repository Documentation
- [x] In-app Legal Screens (About, Privacy Policy, Open Source Licenses)
- [x] JSON Backup & Restore System

## Phase 3: Enhanced Analytics & Export (Upcoming)
- [ ] Export Semester Reports to PDF
- [ ] Shareable Timetable Images
- [ ] Advanced Charts & Graphs for Attendance Trends
- [ ] Exam Countdown Widgets & Timers
- [ ] Localization (Multi-language support)

## Phase 4: Cloud & Sync (Future Concept)
- [ ] Google Drive / iCloud Backup Integration (Opt-in only)
- [ ] Cross-device Synchronization
- [ ] Web Dashboard Companion

*Note: The roadmap is subject to change based on community feedback and contributor availability.*
